# Contribuindo para o Leaf

O Leaf fica feliz que você esteja disposto a contribuir com nossos projetos. Geralmente somos
bastante flexíveis com todos os PRs enviados, mas ainda existem algumas diretrizes que
você pode seguir para tornar o processo de aprovação mais tranquilo.

Para versões 1.21.4+, consulte o [CONTRIBUTING.md](https://github.com/PaperMC/Paper/blob/main/CONTRIBUTING.md) do Paper.

Para versões anteriores, consulte o [CONTRIBUTING.md](https://github.com/PaperMC/Paper-archive/blob/ver/1.21.1/CONTRIBUTING.md) do Paper-archive.
