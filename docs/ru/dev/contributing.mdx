# Участие в разработке Leaf

Leaf рад, что вы готовы вносить вклад в наши проекты. Мы, как правило,
очень лояльно относимся ко всем отправляемым PR, но существуют некоторые рекомендации,
которым можно следовать, чтобы процесс одобрения прошёл более гладко.

Для версии 1.21.4+, смотрите [CONTRIBUTING.md](https://github.com/PaperMC/Paper/blob/main/CONTRIBUTING.md) Paper.

Для более старых версий, смотрите [CONTRIBUTING.md](https://github.com/PaperMC/Paper-archive/blob/ver/1.21.1/CONTRIBUTING.md) Paper-archive.
